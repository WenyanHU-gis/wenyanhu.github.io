Renewables & Heritage WebApps

Web GISETLUI/UX

This project focuses on the development and maintenance of specialized GIS web applications tailored for managing renewable energy sites and heritage data assets across Victoria. The system streamlines complex ETL workflows and provides cross-functional visibility for planning teams.

Key Responsibilities:

  • Designed and deployed ETL (Extract, Transform, Load) workflows to ingest external datasets and standardize them for web use.
  • Configured interactive web maps utilizing the ArcGIS Maps SDK for JavaScript, ensuring smooth rendering and responsive UI/UX.
  • Implemented robust user management systems to control access levels for internal stakeholders and external viewers.
  • Optimized spatial queries and feature retrieval to handle large volumes of heritage polygons dynamically without performance degradation.

Impact:

Delivered streamlined internal tools that allowed planning teams to overlay renewable project boundaries directly against protected heritage zones, cutting manual review times significantly and ensuring regulatory compliance.
