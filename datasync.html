Automated Spatial Data Sync & Pipelines

PythonAWS EC2CI/CDArcGIS Server

This project involves building automated Python-based workflows to synchronize high-priority spatial data between organizational ShareDrives and AWS EC2 infrastructure. The goal is to ensure data consistency across hybrid environments and streamline the web map publishing process.

Key Responsibilities:

  • Developed Python script using the ArcGIS API for Python to automate the publication of Map, Feature, and Map services.
  • Configured and scheduled Python scripts in Jenkins to run automated synchronization and publishing workflows.
  • Implemented robust data synchronization logic to transfer GB-scale spatial datasets securely to AWS instances.
  • Built Git-based pipelines to version control service definitions and automate deployment to staging and production environments.
  • Engineered error-handling and notification systems to alert teams of synchronization status or service interruptions.

Impact:

These improvements increased the reliability and efficiency of service publishing by reducing manual effort, improving version control, and enabling scheduled updates. Keeping ArcGIS project and service definition files under Git also made it easier to track changes, manage publishers, and maintain consistent deployment across environments.
