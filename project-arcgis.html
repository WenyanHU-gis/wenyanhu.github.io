ArcGIS Enterprise Deployment & Development

AWS EC2ArcGIS EnterpriseServer Admin

This project involved deploying a complete ArcGIS Enterprise architecture on AWS EC2 infrastructure. The deployment included ArcGIS Server, Portal for ArcGIS, ArcGIS Data Store, and Web Adaptors.

Key Responsibilities:

  • Managed AWS security groups to ensure secure access to spatial data and services.
  • Migrated legacy spatial datasets to the newly structured Enterprise geodatabases.
  • Published highly available Feature Services and custom Geoprocessing Services for web application consumption.
  • Configured federated servers to work seamlessly with the base deployment.

Impact:

Established a robust, scalable backend that supports thousands of daily maps requests and deep spatial analytics across the organization, improving data governance and minimizing downtime.
