Renewables Summary Page

A high-visibility dashboard for the Department of Transport and Planning, summarizing Victoria's renewable energy generation facilities by status and type.

Key Functions:

  • Area Search: Quickly find and zoom to specific project areas.
  • Dynamic Summaries: Automatic tabulation of capacity (MW) across operating, approved, and Lodged projects.
  • Data Export: Functionality to download data tables for offline analysis.
  • Layer Management: Interactive layer list to toggle specific project visibility.
  • Categorical Tabs: Specific visualization for Solar, Wind, Battery, and Bioenergy datasets.
Technologies: ArcGIS API for JavaScript, Python, Spatial Data Governance
